The tutorial series contained on this platform has been the result of 2 years of research by it's creator,Philip Modinos. The main focus of the tutorials is a deep dive in concepts surrounding the practices of frameworks like laravel and vue.js.
The greatest problem that aspiring web developers face in their search for knowledge is the lack of context. This tutorial series aims to provide that context by providing a step by step guide to the creation of a full stack application.
The final part of the tutorial leverages all the knowledge accumulated from the previous parts to demonstrate how with the guidance of AI alongside the presets of laravel and vue.js, a developer can create a web application that's ready to go in a few hours, always demonstrating the process in it's full context.
Click any of the icons below to get started, or click the about button to learn more about how the tutorial is structured.

Who is this tutorial for?


If you are a complete beginner this tutorial is not for you. It does not cover the absolute basics, like data types, loops, operators etc. It is essential that you have some familiarity with these concepts if you wish to be able to follow this tutorial.



How to use this tutorial:


The tutorial is structured in 4 parts, each part is a series of video tutorials. Click on one of theicons below and you will be taken to that category. Each category has a list of episodes, click on an episode to view the video. The episodes are structured in a way that you can follow along with the video and code along with the instructor. The code for the first category - Classes in PHP is not available on github, as this is a tutorial about concepts and there is no code to retain. The code for the other categories is available on github, you can access the repo from the links provided.
Videos are hosted on youtube, you can access the videos by clicking on the links provided and a new tab will open with the video.
If you have any questions or feedback, please click the contact button and send a message. Enjoy the tutorials!
